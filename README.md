## 게임 데드락의 자막 파일 AI번역을 위한 자동화 코드입니다.  
데드락의 VO자막 파일을 번역하기 위한 목적으로 작성되었지만  
각종 파일에도 사용 가능하며 api 키 값을 사용함으로 원하는 제미나이 모델로 변경 가능합니다.

## 기초 사용법
1. API_KEY를 기입합니다.  
2. INPUT_FILE = "" 에 번역을 원하는 파일의 이름을 작성합니다.  
3. BATCH_SIZE 를 AI 모델의 성능과 리밋에 맞게 조절합니다. (너무 클 경우 누락될 수 있음)
4. prompt 항목에서 지시 사항을 넣고 실행시킵니다.


## 그외
작업을 시작하면서 progress.txt 파일에 이번 request 중 번역을 시작한 라인 넘버를 기록한 후 실행 시 참고하기 때문에  
중간에 오류나 취소 후 이어서 다시 작업할 수 있습니다. (중단 시 API 키를 교체 가능)

번역 된 내용은 OUTPUT_FILE = "" 에 append로 작성됩니다.
